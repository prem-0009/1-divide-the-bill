# 1-divide-the-bill

## 1-first-Node-App

#### Bill Split Calculator

#### Main goal 
>* This app will take:
>  1. Total bill.
>  2. Tip percentage.
>  3. How many ways to split it.
>* The output will be: how much each person owes?

#### Stretch goal.
>* The app can take total bill only too, and leave the tips to be decided by the user.
